# Pixie — macOS Desktop App

System-wide screenshot and annotation tool for macOS.

## Install

1. Unzip **Pixie Desktop.zip**
2. Double-click **Install Pixie.command**
3. Wait ~30 seconds while it builds
4. Drag **Pixie.app** into your Applications folder

That's it. Launch Pixie from Applications like any other app.

> **Requires Node.js** — if you don't have it, the installer will open nodejs.org for you.

## Shortcuts

| Shortcut | Action |
|----------|--------|
| **⌘⇧6** | Capture area — drag to select, then annotate |
| **⌘⇧7** | Instant full-screen capture |

Pixie lives in your menu bar (no Dock icon). Click **⊡** to access the menu.

## After selecting an area (⌘⇧6)

| Action | How |
|--------|-----|
| Annotate | Pencil tool — draw freehand |
| Erase | Eraser tool |
| Change colour | Click a colour dot |
| Toggle corners | Round ↔ Sharp |
| Move selection | Drag inside the selection |
| Copy to clipboard | **Copy** or **⌘C** |
| Save to Desktop | **Save** or **Enter** |
| Cancel | **Escape** |

> macOS will ask for Screen Recording permission on first launch — grant it in  
> System Preferences → Privacy & Security → Screen Recording.
